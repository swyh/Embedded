## Enviroment
- MicroC/OS-II : embedded system에 들어갈 RTOS(Real Time Operating System)

## Requirements
- DosBox 0.74 : OS 실행을 확인할 수 있는 애뮬레이터
- AVR Studio 4 : OS를 Device에 porting하기 위한 개발 환경
